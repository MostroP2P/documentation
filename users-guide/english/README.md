# Users-guide

[https://mostro.network/docs-english](https://mostro.network/docs-english/)

## For contributors

### Requirements

* Install [Just](https://github.com/casey/just) 
```bash
cargo install just
```
### Procedures

**Init**

```bash
just init
```

**To serve the site locally**

```bash
just serve
```

**To build the site**

```bash
just build
```
