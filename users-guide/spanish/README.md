## Para contribuidores

### Requerimientos

* Instala [Just](https://github.com/casey/just) 
```bash
cargo install just
```

### Procedimiento

**Iniciar**

```bash
just init
```

**Para ejecutar el sitio localmente**

```bash
just serve
```

**Para construir el sitio**

```bash
just build
```
